# scripticus (client)

The Scripticus CLI client: publish, discover, version, and install the
scripts your team shares. See the [repository README](../README.md) for the
full picture.

## Development

From the repository root:

```console
$ uv sync                # create/update the workspace environment
$ uv run scripticus -v   # run the CLI
$ uv run pytest          # run the tests
```
