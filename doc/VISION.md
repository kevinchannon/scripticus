# Vision

Scripticus is a package manager and registry for scripts: it gives teams a single, namespaced place to publish, discover, version, and install the scripts they share, with the ergonomics of a modern package manager (`scripticus install <pkg>`) rather than ad hoc copying from wikis, chat, and git repos.

It is designed internal-first — trust comes from organisational access control — but with an architecture (content-addressed artifacts, fully namespaced identity) that does not preclude a future public offering with stronger cryptographic assurance.
